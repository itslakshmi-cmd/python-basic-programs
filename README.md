# Python Basic Programs

This repository contains a collection of Python practice programs to understand core Python concepts.
It is useful for beginners, students, and freshers.

## Concepts Covered
- Variables and input/output
- Swapping numbers without using a third variable
- Conditional statements
- Loops
- Functions and return values
- Default and variable-length arguments
- Local and global variables
- Lists and list comprehensions
- Finding largest and second largest elements
- Removing duplicates from a list
- map() and filter()
- String operations
- Sets and dictionaries
- Regular expressions (email extraction and mobile number validation)
- Basic problem-solving programs

## How to Run
1. Install Python

   git clone https://github.com/itslakshmi-cmd/python-basic-programs.git
